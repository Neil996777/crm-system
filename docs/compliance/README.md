# Compliance Documentation Index

## Purpose

This directory defines project documentation standards for process compliance
and later external review readiness.

Current documents:

- `cmmi-process-standard.md`: project evidence and process-control standard
  aligned to CMMI-style software delivery discipline.
- `evidence-register-template.md`: running evidence ledger template for gates,
  reviews, requirements, design, implementation, QA, integration, audit, and
  release decisions.

## Scope Rule

These documents do not claim CMMI certification or appraisal results. They
define internal readiness evidence only. Formal CMMI appraisal must be handled
through an authorized CMMI appraisal provider and the applicable official model
version.
