# Intellectual Property Documentation Index

## Purpose

This directory defines the CRM project's software copyright and patent-readiness
documentation standards.

Current documents:

- `software-copyright-standard.md`: China software copyright registration
  preparation standard.
- `patent-readiness-standard.md`: invention disclosure, confidentiality, and
  patent-preparation standard.

## Important Rule

Do not commit identity documents, signed application forms, secrets, customer
data, or unpublished patent-sensitive technical details to the public GitHub
repository. Store formal registration packages and invention disclosures in a
private controlled location.
